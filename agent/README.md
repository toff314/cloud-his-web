# 智慧云门诊官网

这是一个面向医院、中医馆的在线服务平台官网，提供产品介绍、解决方案、客户案例、关于我们、免费试用、版本套餐等功能。

## 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite
- **路由**: Vue Router 4
- **数据存储**: LocalStorage (模拟SQLite)
- **样式**: 原生CSS

## 功能特性

### 顶部导航栏
- 左侧显示"智慧云门诊"Logo
- 中部导航菜单：首页、产品介绍（含下拉菜单）、解决方案、客户案例、关于我们
- 右侧操作区：免费试用、版本套餐、登录按钮
- 固定于页面顶部

### 页面功能

#### 首页
- 轮播图区域（支持自动轮播）
- 产品介绍预览（显示3个产品模块）
- 解决方案预览（显示5个解决方案按钮及展示）
- 客户案例预览（显示3个客户案例）

#### 产品介绍
- 网格布局展示6个功能模块
- 每个模块包含圆形占位图与文字标题描述

#### 解决方案
- 横排并排排列的5个解决方案按钮
- 点击按钮切换中间大图展示和下方描述

#### 客户案例
- 3排3列展示客户案例
- 每个案例包含医院Logo占位图和医院名称及描述

#### 关于我们
- 公司简介区域：左侧文案，右侧公司图片占位
- 荣誉资质区域：4个荣誉证书展示单元
- 公司愿景区域：3个愿景详情（图片在上，文字在下）

#### 免费试用
- 左右分栏设计，移动端上下堆叠
- 表单包含：手机号（必填）、验证码（带获取验证码按钮）、诊所名（必填）、联系人（必填）、邀请码（选填）
- 隐私承诺声明

#### 版本套餐
- 横向可滑动的套餐卡片容器
- 包月套餐：￥100元，30天有效期
- 包年套餐：￥200元，365天有效期
- 套餐功能对比表格

#### 登录
- 账号密码登录表单
- 记住账号选项
- 忘记密码链接

### 页脚
- 左侧：ICP备案、公司名称、公司地址
- 右侧：微信公众号二维码占位

### 右侧悬浮导航
- 电话联系、客服微信、免费试用、立即登录
- 微信弹窗显示客服微信二维码

## 数据存储

项目使用LocalStorage模拟SQLite数据库存储，所有数据在首次加载时初始化。

### 数据表结构

- **banner**: 轮播图数据
- **product**: 产品模块数据
- **solution**: 解决方案数据
- **customer**: 客户案例数据
- **honor**: 荣誉证书数据
- **vision**: 公司愿景数据
- **package_plan**: 套餐数据
- **system_config**: 系统配置数据

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```
